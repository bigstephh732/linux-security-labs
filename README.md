# Linux Security Labs

This repository contains hands-on cybersecurity labs performed in a Linux
(Ubuntu) virtual environment using VMware Fusion.

## Lab Environment
- Host OS: macOS
- Guest OS: Ubuntu Linux (ARM)
- Virtualization: VMware Fusion
- Tools: Nmap

## Labs Included
1. Basic TCP Port Scanning
2. Service & Version Detection
3. UDP Port Scanning
4. OS Detection & Aggressive Scanning
5. Nmap Scripting Engine (NSE)

## Purpose
These labs demonstrate foundational skills in network reconnaissance,
service enumeration, and security analysis using ethical and controlled
scanning techniques.

## Disclaimer
All scans were performed against localhost (127.0.0.1) in a controlled
lab environment. No external systems were targeted.
